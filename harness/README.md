# Harness package

`harness/research-project-os/` contains the current Codex skill implementation of the generic `.project_os/` harness:

- `SKILL.md`: agent trigger and progressive-disclosure instructions;
- `scripts/project_os.py`: CLI entry point;
- `scripts/_*.py`: implementation modules;
- `references/`: schema and policy references;
- `templates/project_os/`: scaffold written into projects.

This repository documents the generic harness. The skill library can mirror or package this directory as an agent-facing skill.
